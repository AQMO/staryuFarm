# IDEA 启动指南

## 解决 "Unresolved plugin: maven-shade-plugin" 报错

### 方法1：配置阿里云 Maven 镜像（推荐）

1. 打开 IDEA → Settings → Build → Build Tools → Maven
2. 找到 `User settings file`，点击 Override，创建/编辑文件 `~/.m2/settings.xml`
3. 写入以下内容：

```xml
<?xml version="1.0" encoding="UTF-8"?>
<settings xmlns="http://maven.apache.org/SETTINGS/1.2.0"
          xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
          xsi:schemaLocation="http://maven.apache.org/SETTINGS/1.2.0 https://maven.apache.org/xsd/settings-1.2.0.xsd">
  <mirrors>
    <mirror>
      <id>aliyunmaven</id>
      <mirrorOf>central</mirrorOf>
      <name>阿里云公共仓库</name>
      <url>https://maven.aliyun.com/repository/public</url>
    </mirror>
    <mirror>
      <id>aliyunmaven-spring</id>
      <mirrorOf>spring</mirrorOf>
      <name>阿里云Spring仓库</name>
      <url>https://maven.aliyun.com/repository/spring</url>
    </mirror>
  </mirrors>
</settings>
```

4. 点击 Apply → OK
5. 右键项目 → Maven → Reload Project

### 方法2：手动安装插件

```bash
# 在终端执行
mvn dependency:resolve-plugins
```

### 方法3：IDEA中忽略shade插件错误

如果只需开发调试，shade插件报错不影响运行，可以忽略。

## IDEA 启动步骤

### 1. 导入项目
- File → Open → 选择项目根目录（含pom.xml的目录）
- 等待Maven自动下载依赖（首次较慢）

### 2. 配置数据库
确保 MySQL 已启动，创建数据库：

```sql
CREATE DATABASE IF NOT EXISTS staryu_farm DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

建表SQL在项目根目录的 `init-db.sql` 文件中。

### 3. 配置数据源
编辑 `src/main/resources/jdbc.properties`：

```properties
jdbc.driver=com.mysql.cj.jdbc.Driver
jdbc.url=jdbc:mysql://localhost:3306/staryu_farm?useSSL=false&serverTimezone=Asia/Shanghai&characterEncoding=UTF-8&allowPublicKeyRetrieval=true
jdbc.username=root
jdbc.password=你的密码
```

### 4. 启动项目
**方式A：直接运行 Main 类（推荐开发使用）**
1. 打开 `src/main/java/com/staryu/Main.java`
2. 右键 → Run 'Main.main()'
3. 控制台出现 `星语农庄服务启动成功！端口: 5000` 即启动成功

**方式B：Maven 命令启动**
```bash
mvn package -DskipTests
java -jar target/staryu-web.jar --server.port=5000
```

### 5. 访问地址
| 服务 | 地址 |
|------|------|
| 管理后台 | http://localhost:5000/admin/dashboard |
| 房间API | http://localhost:5000/api/rooms |
| 菜品API | http://localhost:5000/api/food |
| 农产品API | http://localhost:5000/api/products |
| 果木API | http://localhost:5000/api/fruit-trees |
| 地块API | http://localhost:5000/api/plots |
| 订单API | http://localhost:5000/api/orders |
| 配置API | http://localhost:5000/api/config |
| 统计API | http://localhost:5000/api/stats |

### 6. staryu-uniapp 前端启动
```bash
cd staryu-uniapp
pnpm install
pnpm run dev
```
访问 http://localhost:5173

## 常见问题

### Q: Maven依赖下载慢/失败？
A: 配置阿里云镜像（见方法1）

### Q: 端口5000被占用？
A: 修改启动参数 `--server.port=8080`，或在Main类VM options中添加 `-Dserver.port=8080`

### Q: MySQL连接失败？
A: 检查MySQL是否启动、账号密码是否正确、数据库staryu_farm是否存在

### Q: 管理后台页面空白？
A: 检查浏览器控制台Network，确认API请求是否正常返回
